康 佳

96 年程序员，现居天津。纯音乐爱好者，书籍爱好者（PS：喜欢买但不一定看）。半个断舍离主义者，认为整理东西是最解压的方式之一。

喜欢在周末逛个超市，做点美食。偶尔小酌一杯，有为了抛开重重心事安静无争的入睡，有为了麻痹某种境况下的胆怯。 偶尔打打游戏，一直在辅助的位置尝试着所有自认为有趣（快乐）的英雄，信奉快乐游戏的前提是：进入游戏的第一件事，键入 /mute all，再按下 Enter。 饿的时候只想吃咸香的。不习惯穿拖鞋。不喜欢吃饼干。

自认为是个有情怀的人儿，会经常记录一些个人的所见所闻所思，你可以在我的博客驻足聆听我的现在与过往。 喜欢极客精神，对各种技术怀有敬畏，当下正专注于 Java Web 开发，欢迎来我的技术笔记了解更多我的技术栈。 如果恰巧你在招贤纳士，不妨瞄一眼我的简历。 或者你想联系我，可以给我发邮件。
